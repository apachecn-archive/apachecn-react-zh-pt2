## 地理定位

在本节中，您将了解如何通过 React 本机应用使用 iOS 定位服务。定位服务在许多流行的应用中使用非常频繁，尤其是在旅行、导航、出租车共享和不胜枚举的应用中。这个特性极大地改善了用户体验，令人欣慰的是它非常容易实现。

当我们用`react-native init`设置一个项目时，默认情况下 React Native 中的地理位置属性是启用的。让我们创建一个应用来实现这一功能:

## 本地警报
